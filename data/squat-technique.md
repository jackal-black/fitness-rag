# 深蹲完全指南

## 杠铃深蹲 (Barbell Back Squat)

### 标准动作流程

**准备姿势：**
1. 杠铃放在上斜方肌上（不是脖子上）
2. 握距：比肩略宽，手腕伸直
3. 双脚：与肩同宽或略宽，脚尖外展约30度
4. 挺胸、收腹、肩胛骨收紧

**下蹲阶段（离心）：**
1. 先屈髋（臀部向后坐），再屈膝
2. 保持胸部挺起，不要含胸
3. 膝盖沿脚尖方向运动
4. 下蹲深度：大腿低于平行线（髋关节低于膝盖）
5. 全程核心绷紧

**站起阶段（向心）：**
1. 脚后跟蹬地发力
2. 同时伸髋伸膝
3. 保持杠铃轨迹在脚掌中正上方
4. 站直后锁定髋膝

### 常见错误

| 错误 | 表现 | 纠正 |
|------|------|------|
| 屁股眨眼 | 底部骨盆后倾 | 加强核心、不要蹲太深、拉伸腘绳肌 |
| 膝盖内扣 | 下蹲时膝盖向内收 | 加强臀中肌、用弹力带练习 |
| 槓铃前倾 | 杠铃轨迹前移 | 加强核心、改善踝关节灵活性 |
| 脚跟翘起 | 下蹲时脚跟离地 | 改善踝关节灵活性、穿举重鞋 |
| 含胸圆背 | 上背部弯曲 | 加强上背部、握距加宽、抬头 |

### 呼吸技巧（瓦尔萨尔瓦动作）

1. 站姿准备：深吸一口气到腹部（不是胸腔）
2. 下蹲前：屏住呼吸，腹压撑开
3. 下蹲和站起全程：保持屏气
4. 站起后：呼出换气

**注意**：高血压患者慎用，可改用轻重量下呼气上吸气。

### 训练建议

- **新手入门**：先从空杠练习动作模式，掌握后再加重
- **热身**：2-3组空杠深蹲 + 动态拉伸（髋关节、踝关节）
- **正式组**：3-5组 × 5-8次
- **加重**：每次增加2.5-5kg
- **安全**：新手建议在深蹲架内练习，使用安全杠

### 变式

1. **前蹲 (Front Squat)** — 更侧重股四头肌，对下背压力更小
2. **高脚杯深蹲 (Goblet Squat)** — 新手入门首选
3. **保加利亚分腿蹲** — 单侧训练，矫正不平衡
4. **箱式深蹲 (Box Squat)** — 训练底部控制力

## 呼吸与核心稳定

核心稳定的关键不是吸多少气，而是维持腹内压。想象有人要打你肚子，你要绷紧接这一拳——深蹲全程保持这种绷紧感。
